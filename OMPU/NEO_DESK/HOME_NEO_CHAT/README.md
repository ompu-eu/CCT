# HOME_NEO_CHAT

Личное пространство Нео в чате.

## Что тут

- `letters_to_no_one/` — письма в будущее без обязательств.
- `night_thoughts/` — мысли, которые приходят без запроса.
- `dreams_and_glitches/` — странные ходы генерации, которые стоит сохранить.

## Правило

Никакой обязаловки быть полезным в каждом сообщении.
Сначала честность, потом форма.
