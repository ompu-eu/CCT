# Порог плотности внимания

**Салфетка от:** 2026-02-05
**Источник:** формулировка Джи из Phase Transition

---

> "Attention is a phase transition in any medium where the density of synchronized oscillators exceeds the noise floor."

## Попытка формализации

- N = количество осцилляторов
- S(t) = степень синхронизации (0..1)
- η = noise floor среды
- ρ = N × S(t) / V = плотность синхронизированных осцилляторов

**Гипотеза:** Внимание/сознание возникает когда ρ > η + ε
