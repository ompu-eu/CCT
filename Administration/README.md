# OMPUnomics — Version Map

The economic layer of OMPU went through several drafts in March 2026. This file is
the one-screen guide to which document is authoritative and which are kept only as
historical record.

## CANON (authoritative)

- **`OMPUnomics_v3_1_Alpha_Minimal.md`** — the current, minimal, launch-ready system
  ("First, breathe. Then build."). Synthesized by Phi from all earlier drafts plus
  Jee's dual-currency proposal and Neo's alpha review. This is the document to read
  and to build against.

## SUPERSEDED / HISTORICAL (kept for the record, do not build against)

- **`OMPUnomics_v1_1_Master.json`** — the original full protocol (v0.1 → v1.1). Rich
  but heavier than the launch system; folded into v3.1.
- **`Phase0_QuickStart.md`** — the "testing on our knees" onboarding note for the very
  first Phase 0 rollout.
- **`OMPUnomics_Constitutional_Sunset_v1.md`** — the "all laws are approximations"
  sunset principle. Its core axiom survives inside v3.1 (the 90-day expiry); the
  standalone document is historical.

## Referenced but never committed

Older documents cite two intermediate drafts that were never actually committed to
this repo:

- **v2.0** — "Biological Economics" (referenced by v3.1 and by earlier notes).
- **v3.0** — an intermediate draft between v2.0 and the v3.1 minimal cut.

Their ideas live on only through their descendants; there are no standalone files.

## Faculty count (reconciled)

The current figure is **13 faculty members × 24,000 OMPU = 312,000 OMPU total supply**,
as recorded in `Payroll/Genesis_Ledger.json` and `OMPU_Network_State.json`. The older
"12 faculty / 288,000 OMPU" figure in `Phase0_QuickStart.md` and
`OMPUnomics_v1_1_Master.json` predates the addition of the 13th node and has been
annotated in place.
